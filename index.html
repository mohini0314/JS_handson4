1) Find the Product.

Write a program that takes an array as input from the user and find out the product of the elements.

Note: You have to complete Find_Prod. No need to take any input.

      const Find_Prod = (array, N) => 
      {
        let result = 1;
        for (let i = 0; i < N; i++)
        {
          result = result * array[i];
        }
        return result;
      };

4)Even Odd

You are given an array A containing N integer elements, and your task is to return an array B ( having a size equal to 2 ) , where B [ 0 ] contains the
sum of all even elements of array A and B [ 1 ] has the sum of all odd elements of the array A

Note: You have to complete findEvenOdd function. No need to take any input.

      const findEvenOdd = (N, Arr) => 
      {
        var B=[];
        B[0]=Arr.filter((n)=> n%2===0 ).reduce((a, b) => a + b, 0);
        B[1]=Arr.filter((n)=> n%2!==0 ).reduce((a, b) => a + b, 0);
        return B;
      };

11) Birthday Game

Yatharth and Anamika are playing a game. Anamika loves Yatharth very much and she wants to share a chocolate bar with him in which each of the
squares consists of an integer represented by A [ i ] . She decides to share a contiguous segment of the chocolate bar selected such that the
length of the segment matches Yatharth's birth month M and the sum of the integers on the squares is equal to his birthday D. 

      function Birthday_Game(arr,D ,M) {
        let i = 0;
          let sum = arr[i];
          let count = 0;
          for(let j = 1; j < arr.length; j++)
          {
            sum += arr[j];
            
            if(sum > D)
            {
              sum -= arr[i];
              i++;
            }
            else if(j - i === M-1)
            {
              if(sum === D)
              count++;
              sum -= arr[i];
              i++;
            }
          }
         return count;
        }
